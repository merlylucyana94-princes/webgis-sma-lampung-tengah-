Tentang WebGIS Ini

WebGIS Persebaran SMA Negeri Kabupaten Lampung Tengah adalah sistem informasi geografis berbasis web yang dirancang dan dikembangkan sebagai tugas akhir mata kuliah Perancangan WebGIS, Program Studi Pendidikan Geografi, Universitas Lampung.

🔖 Latar Belakang
🎯

Permasalahan Akses Pendidikan

Kabupaten Lampung Tengah sebagai salah satu kabupaten terluas di Lampung memiliki tantangan pemerataan akses fasilitas pendidikan. Pemetaan spasial menjadi kunci solusinya.

💡

Pentingnya Visualisasi Spasial

Data berbentuk tabel tidak cukup untuk memahami pola distribusi geografis. WebGIS hadir untuk mentransformasi data tabuler menjadi visualisasi peta yang informatif.

🗺️

Solusi Berbasis Web

WebGIS berbasis browser dipilih karena aksesibilitasnya yang tinggi — dapat diakses tanpa instalasi perangkat lunak khusus, cukup dengan browser modern.

🎯 Tujuan & Manfaat
1
Menyediakan visualisasi interaktif persebaran SMA Negeri di seluruh Kabupaten Lampung Tengah.
2
Membantu masyarakat, orang tua, dan siswa menemukan informasi sekolah terdekat dengan mudah.
3
Mendukung pengambilan keputusan pemerintah daerah terkait pembangunan fasilitas pendidikan.
4
Mengidentifikasi wilayah yang masih kekurangan akses SMA Negeri untuk perencanaan masa depan.
5
Sebagai implementasi nyata pembelajaran mata kuliah Perancangan WebGIS Prodi Pendidikan Geografi.
🛠️ Teknologi yang Digunakan
Stack teknologi open-source yang mendukung WebGIS ini
🍃
Leaflet.js
Library peta interaktif open-source berbasis JavaScript untuk rendering WebGIS
📐
GeoJSON
Format standar pertukaran data geospasial berbasis JSON untuk menyimpan koordinat dan atribut sekolah
🌐
OpenStreetMap
Sumber data peta dasar (basemap) open-source yang digunakan sebagai latar belakang peta
💻
HTML5 / CSS3 / JS
Teknologi web standar untuk membangun antarmuka pengguna yang responsif dan interaktif

🔬 Metodologi
1️⃣Pengumpulan data atribut dan spasial sekolah
2️⃣Konversi data ke format GeoJSON
3️⃣Perancangan antarmuka WebGIS berbasis Leaflet.js
4️⃣Integrasi data spasial dengan visualisasi peta interaktif
5️⃣Analisis distribusi dan penyusunan laporan
